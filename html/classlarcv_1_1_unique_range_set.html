Detailed Description

template<class T>
class larcv::UniqueRangeSet< T >

std::set of larcv::Range, which does not allow any overlap in contained element.

std::set<Range> w/ modified insert/emplace function. Original std::set does not allow
modification of element. I assume what we're interested in is "find if the range already \n exists, and merge if it exists". The insert function does that by recursively looking up
overlapping elements w.r.t. input argument of insert function.
One important function worth noting is larcv::UniqueRangeSet::Exclusive which takes two
input arguments, "start" and "end", and returns larcv::UniqueRangeSet of all exclusive
regions between "start" and "end". By definition, merging this return with the original
instance will result in 1 huge larcv::Range.
