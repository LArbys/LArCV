Detailed Description

template<class T>
class larcv::Range< T >

represents a "Range" w/ notion of ordering.

A range is defined by a pair of "start" and "end" values. This is stored in std::pair
attribute larcv::Range::_window. This attribute is protected so that the start/end cannot
be changed w/o a check that start is always less than end. Note the specialization
requires a template class T to have less operator implemented.
