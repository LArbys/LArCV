Detailed Description

Factory class for instantiating process instance This factory class can instantiate a specified process instance w/ provided instance name.


The actual factory core (to which each algorithm must register creation factory instance) is
a static std::map. Use static method to get a static instance (larcv::ProcessFactory::get)
to access a factory.
